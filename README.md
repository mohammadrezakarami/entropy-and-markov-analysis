# Entropy and Markov Analysis

This repository contains a set of assignments, implementations, and reports related to entropy, Markov processes, quantization, and related information-theoretic topics.

## Repository Structure

- `1/`: entropy-related material and MATLAB implementation.
- `2/`: GUI and report files.
- `3/`: MATLAB assignment work.
- `4/`: notebook-based work and report.
- `5/`: GUI-based assignment files.
- `6/`: MATLAB implementation and report.
- `7/`: LBG-related material and report.

## Topics Covered

- discrete entropy
- Markov analysis
- quantization
- Lloyd and LBG algorithms
- MATLAB and notebook-based experimentation

## Getting Started

1. Clone the repository.
2. Use MATLAB for `.m` and `.mlapp` files.
3. Use Jupyter for `.ipynb` files.
4. Open the PDF documents for assignment descriptions and reports.

## Notes

This repository is organized as a multi-part coursework submission across several related topics in information theory and stochastic analysis.
